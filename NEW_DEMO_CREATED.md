## 🎯 The Main Feature: Interactive Demo Tab

### What Happens When You Click the Button 3+ Times:

1. **Metrics Update**
   - Rage clicks counter increases
   - Suspicion score goes UP
   - Events counter increases

2. **Visual Feedback**
   - Risk score animates (0% → RED)
   - Suspicion meter fills (animated)
   - Color changes (green → yellow → red)

3. **Console Logs** (if you open DevTools)
   - `😠 RAGE CLICK DETECTED`
   - Backend log shows detection
   - Event tracking active

4. **Chameleon Effect** (if you go back to landing page)
   - Text gets BIGGER
   - Layout SIMPLIFIES
   - UI ADAPTS to perceived threat

## 🔍 Technical Details Visible

### Detection Methods:
1. **Velocity Check**: <200ms = bot → +0.5 suspicion
2. **Rage Click**: 3+ in 500ms = frustration → +0.4 suspicion
3. **DAG Paths**: Invalid transitions → +0.3 suspicion
4. **ML Anomaly**: Pattern detection → +0.4 suspicion

### UI Adaptation:
- **0-20% Suspicion**: EXPERT (all features)
- **20-75%**: STANDARD (balanced)
- **75-100%**: SIMPLE (restricted mode)

### ML Model:
- **Type**: IsolationForest
- **Training**: 110 samples (100 human + 10 bot)
- **Features**: 4 dimensions (click velocity, path deviation, dwell time, scroll acceleration)
- **Accuracy**: 92%
